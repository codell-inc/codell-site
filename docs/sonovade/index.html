Core Functions

現地調査から書類作成まで、
ひとつの流れに。

案件情報と施工内容を一度入力すれば、図面・写真・部材・金額を同じ案件の中で管理できます。

  1. 01

    案件登録

    利用者情報や現場情報を登録し、住宅改修に必要な情報を案件ごとにまとめます。

  2. 02

    図面作成

    スマホ・タブレットの直感的な操作で、現地調査中に間取りを作成できます。

  3. 03

    改修箇所入力

    図面上に手すりや改修箇所、寸法を入力し、施工内容を分かりやすく可視化します。

  4. 04

    部材・原価計算

    商品を選ぶだけで必要な部材、数量、金額を反映。変更にもすぐ対応できます。

  5. 05

    写真管理

    改修前後の写真を案件や施工箇所に紐づけ、写真台帳の作成へつなげます。

On Site Value

ソノバデを使うメリット

  1. 01

    スマホ・タブレットで
    図面を簡単作成

    直感的な操作で、現地調査中に図面を作成。紙へ書いた情報を帰社後に転記する手間を減らします。

  2. 02

    施工場所を入れるだけで、
    見積書・申請書類へ

    施工箇所と寸法を入力すると、必要な部材と金額を案件へ反映。書類作成の手間とタイムラグを短縮します。

  3. 03

    写真を施工箇所ごとに、
    分かりやすく整理

    改修前後の写真を図面上の施工箇所へ紐づけて管理。写真台帳や申請書類の準備へつなげます。

  4. 04

    商品を選ぶだけで、
    金額を即座に反映

    部材や商品の変更も選び直すだけ。その場で概算を確認し、お客様との相談を進められます。
